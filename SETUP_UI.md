# Setting Up the Owney Test UI

This guide walks you through building and running the test UI for the Owney mailserver.

## Quick Start

### 1. Install UI Dependencies

```bash
cd ui
npm install
```

### 2. Build the UI

```bash
npm run build
```

This builds the React app to `crates/owney-api/static/` which the Rust server will serve.

### 3. Run the Server

The server will automatically serve the UI at `/` when you run it:

```bash
cargo run --release
```

Then open http://localhost:8008 in your browser.

## Development Workflow

### Option A: Build UI Before Running Server

1. Build the UI: `cd ui && npm run build`
2. Run the server: `cargo run --release`
3. Open http://localhost:8008

### Option B: Simultaneous Development (Recommended)

Run these in separate terminals:

**Terminal 1 - Watch UI changes:**
```bash
cd ui
npm run dev
```

This starts Vite dev server at http://localhost:5173

**Terminal 2 - Run the Rust backend:**
```bash
cargo run --release
```

**To test:** 
- Visit http://localhost:5173 to test UI changes immediately
- The UI dev server proxies JMAP calls to http://localhost:8008

(Note: Update `ui/vite.config.ts` with a proxy config if needed)

## Deploying to Production

1. Build the UI: `cd ui && npm run build`
2. Build the Rust server: `cargo build --release`
3. Run: `./target/release/owneyd` (or your binary name)
4. The server serves the UI at `http://your-server`

## Adding New Features

To add new feature testers (e.g., Contacts, Tasks):

1. Create a new component in `ui/src/components/YourFeature.tsx`
2. Import it in `ui/src/App.tsx`
3. Add a tab button for it
4. Use the `callJmap()` helper from `api.ts` to make JMAP calls

Example:

```tsx
import { useState } from 'react'
import { callJmap, getSession } from '../api'

export default function YourFeatureTester() {
  const [status, setStatus] = useState('')
  const [response, setResponse] = useState('')
  
  const handleGetData = async () => {
    try {
      const session = await getSession()
      const accountId = Object.keys(session.accounts)[0]
      const result = await callJmap([
        ['YourFeature/get', { accountId }, '0'],
      ])
      setStatus('✓ Success')
      setResponse(JSON.stringify(result, null, 2))
    } catch (error) {
      setStatus(`✗ ${error.message}`)
    }
  }

  return (
    <div>
      <button onClick={handleGetData}>Get Data</button>
      {status && <div className="status">{status}</div>}
      {response && <div className="response-box">{response}</div>}
    </div>
  )
}
```

## Troubleshooting

### "Cannot find static directory" error

The server looks for static files in `crates/owney-api/static/`. Make sure you've run:
```bash
cd ui && npm run build
```

### CORS or authentication errors

1. Make sure you have a valid JMAP token
2. Set it in the UI (localStorage is used)
3. Check that the server is running on http://localhost:8008

### Port already in use

If port 8008 is busy:
```bash
LISTEN_ADDR=127.0.0.1:9999 cargo run --release
```

Then open http://localhost:9999
